# AGENTS.md

## 프로젝트 개요
- 이 프로젝트는 Streamlit 기반 기본 챗봇 시작 프로젝트다.
- 메인 실행 파일은 `app.py`다.
- 환경 변수는 `.env` 파일에 저장한다.

## 작업 규칙
- 앱은 단순하고 로컬에서 바로 실행 가능하게 유지한다.
- 비밀값은 `.env`에서 불러오고, 실제 토큰은 소스 코드에 직접 쓰지 않는다.
- UI는 Streamlit으로 구성하고, 프로젝트가 커지기 전까지 챗봇 로직은 `app.py` 중심으로 유지한다.
- 큰 리팩터링보다 작고 목적이 분명한 수정 우선.
- 가능하면 Streamlit 앱 실행으로 변경 사항 확인한다.

## 코드 작성 가이드
- 변수명은 명확하게 작성한다.
- 함수는 짧고 역할이 분명하게 유지한다.
- 로직이 명확하지 않을 때만 짧은 주석 추가한다.
- 의존성은 최소한으로 유지한다.
- 요구사항 없으면 기존 동작 유지한다.

## 보안 규칙
- 실제 API 키나 토큰은 커밋하지 않는다.
- `.env` 값은 민감 정보로 취급한다.
